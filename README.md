# Production Grade DevOps Platform

## Project Overview

This repository contains a Production Grade DevOps Platform built using modern DevOps practices.

The goal of this project is to simulate a real-world enterprise infrastructure running on AWS using Infrastructure as Code (Terraform), Configuration Management (Ansible), Kubernetes (Amazon EKS), GitHub Actions, and GitOps with ArgoCD.

## Technology Stack

- AWS
- Terraform
- Ansible
- Docker
- Kubernetes
- Helm
- ArgoCD
- GitHub Actions
- Git
- Linux
